# WanderQuest

Alpha work in progress AR adventure RPG for RayNeo X3 Pro glasses.

Walk through the real world to collect shards and ingredients, discover
beacons and portals, and fend for yourself against minions while working to
restore justice to the world. WanderQuest is an early experimental build:
systems, visuals, balance, and content are still in motion.

Gameplay clip: https://youtu.be/RAet8Xiiuyk

## Install

Download `WanderQuest.apk` from the latest release and install with:

```bash
adb install -r WanderQuest.apk
```
